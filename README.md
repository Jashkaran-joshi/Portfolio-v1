# Modern Cyber Portfolio

A modern, single-page portfolio website designed for a Full-Stack Developer & Cybersecurity Enthusiast. Built with React.js (Frontend) and Node.js + Express.js (Backend).

## Features

- **Cybersecurity Theme**: Dark mode, neon accents, glitch effects, and terminal aesthetics.
- **Single Page Application**: Smooth scrolling navigation.
- **Responsive Design**: Fully responsive for mobile, tablet, and desktop.
- **Animations**: Powered by Framer Motion.
- **Backend API**: Secure contact form handling with Express.js.
- **Security**: Rate limiting, Helmet headers, and input validation.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Framer Motion, React Scroll, React Icons.
- **Backend**: Node.js, Express.js, Cors, Helmet, Express Rate Limit, Express Validator.

## Getting Started

### Prerequisites

- Node.js installed.

### Installation

1.  **Clone/Open the repository**:
    Navigate to `d:\WorkSpace\Portfolio\My\ModernPortfolio`.

2.  **Setup Backend**:
    ```bash
    cd server
    npm install
    ```
    - Configure `.env` file in `server` directory if needed (e.g., email credentials).

3.  **Setup Frontend**:
    ```bash
    cd client
    npm install
    ```

### Running the Project

1.  **Start the Backend**:
    ```bash
    cd server
    node server.js
    ```
    The server runs on `http://localhost:5000`.

2.  **Start the Frontend**:
    Open a new terminal.
    ```bash
    cd client
    npm run dev
    ```
    The frontend runs on `http://localhost:5173`.

## Customization

- **Personal Data**: Edit `client/src/data.js` to update your information, skills, and projects.
- **Styles**: Edit `client/tailwind.config.js` or `client/src/index.css` for theme changes.
